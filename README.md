# 🏦 Bank Management System (C++-OOP)

A simple **Bank Management System** built using **C++ and Object-Oriented Programming (OOP)** concepts.
The system allows managing bank clients, transactions, users, and currency operations through a console-based interface.

---

## 📌 Features

### 👥 Client Management

* Add new clients
* Update client information
* Delete clients
* Search for clients
* Display all clients list

### 💰 Transactions

* Deposit money
* Withdraw money
* Transfer money between clients
* Show total balances
* View transfer history

### 👤 User Management

* Add new users
* Update users
* Delete users
* User permissions system
* Login system with authentication

### 💱 Currency System

* List all currencies
* Currency exchange calculator
* Convert between currencies

---

## 🧠 Concepts Used

This project demonstrates many important **C++ programming concepts**:

* Object Oriented Programming (OOP)
* Classes & Objects
* Inheritance
* Encapsulation
* File Handling
* Vectors
* Data Validation
* Console UI design

---

## 🗂 Project Structure

Example of some main classes used in the project:

* `clsBankClient` → Manage client accounts
* `clsUser` → Manage system users
* `clsScreen` → Base class for screens
* `clsTransactionsMenueScreen` → Transactions menu
* `clsCurrency` → Currency management
* `clsUtil` → Utility functions
* `clsInputValidate` → Input validation

---

## 📂 Data Storage

The system stores data using **text files**, for example:

* `Clients.txt`
* `Users.txt`
* `Currencies.txt`
* `TransferLog.txt`
